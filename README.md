# ultimos-rastros
Jogo 2D de aventura, plataforma e puzzle narrativo onde você controla o Curupira, um espírito da floresta que está sendo esquecido. Explore ambientes, recupere memórias e restaure seus poderes enquanto descobre o destino da natureza em uma experiência atmosférica inspirada no folclore brasileiro. jogo feito para materia de Desenvolvimento de jogos
